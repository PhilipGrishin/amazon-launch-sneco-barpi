# Documents Checklist — SnEco

## Public Repository Rule

This repository is public. Do not include private certificates, supplier data, confidential costs, personal data, private document links, credentials, or non-public commercial strategy.

Use public-safe status notes only. Store actual private documents outside this repository.

## Active Scope

Only these SnEco SKU are active for the current Amazon Europe launch evaluation:

- SnEco Cheddar 28g EN
- SnEco Gouda 28g EN

Do not use Slovak front packaging, Ukrainian packaging, other cheese variants, nut mixes, or TM Barpi in the current active SnEco scope.

All requirements are preliminary until Amazon Europe marketplace and category checks are completed for amazon.es, amazon.it, amazon.fr, and amazon.de.

Actual requirements depend on Amazon marketplace, category, local regulation, fulfillment model, and Amazon Seller Central requests.
