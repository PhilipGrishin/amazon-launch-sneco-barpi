# Amazon Launch — SnEco & TM Barpi

Public workspace for planning an Amazon launch strategy for SnEco and TM Barpi.

## Project Purpose

This repository organizes non-confidential project materials for Amazon marketplace research, listing drafts, compliance review, launch planning, and decision tracking.

Use placeholders only. Do not add private supplier terms, costs, customer data, proprietary formulas, legal documents, credentials, or any other confidential business information.

## AI Roles

- ChatGPT: strategy partner for planning, positioning, workflows, and decision framing.
- Claude: long-form research and review analysis support, especially competitor reviews and customer language synthesis.
- Codex: repository operator for file structure, templates, version control, and technical workspace maintenance.

## Public Repository Warning

This repository is public. Assume every file can be read by competitors, suppliers, customers, and Amazon. Keep all sensitive details outside this repository.

## Project Workflow

1. Capture placeholder product data and open questions.
2. Track competitor ASINs and public observations.
3. Analyze public reviews and extract customer language.
4. Model unit economics using placeholder inputs until approved values are available in a private system.
5. Draft listings and run compliance checks before using any claims.
6. Plan marketplace selection, PPC, inventory, and launch steps.
7. Record decisions in the decisions log with status and rationale.
